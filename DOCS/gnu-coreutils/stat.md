# stat 命令

## 命令概述

`stat` 命令用于显示文件或文件系统的详细状态信息。它可以显示文件的元数据，包括大小、权限、所有者、时间戳等信息。

**命令格式：**
```
stat [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-L` | `--dereference` | 跟随符号链接 |
| `-f` | `--file-system` | 显示文件系统状态而非文件状态 |
| `-c` | `--format=FORMAT` | 使用指定格式输出 |
| `--printf=FORMAT` | | 类似 `--format`，但解释反斜杠转义 |
| `-t` | `--terse` | 以简洁格式显示信息 |
| `--help` | | 显示帮助信息并退出 |
| `--version` | | 显示版本信息并退出 |

## 参数详细说明

### `-L, --dereference`
跟随符号链接，显示链接指向的文件信息而不是链接本身的信息。

### `-f, --file-system`
显示文件所在文件系统的状态信息，而不是文件本身的信息。显示的信息包括文件系统类型、块大小、总空间、可用空间等。

### `-c, --format=FORMAT`
使用指定的格式字符串输出信息。格式字符串中可以使用以下格式说明符：
- `%a`：八进制表示的访问权限
- `%A`：人类可读的访问权限
- `%b`：分配的块数
- `%B`：以字节为单位的块大小
- `%C`：SELinux 安全上下文字符串
- `%d`：十进制设备号
- `%D`：十六进制设备号
- `%f`：十六进制表示的原始模式
- `%F`：文件类型
- `%g`：所有者的组 ID
- `%G`：所有者的组名
- `%h`：硬链接数
- `%i`：inode 号
- `%m`：挂载点
- `%n`：文件名
- `%N`：带箭头的文件名（如果是符号链接）
- `%o`：I/O 块大小
- `%s`：总字节数
- `%t`：十六进制的主要设备类型
- `%T`：十六进制的次要设备类型
- `%u`：所有者的用户 ID
- `%U`：所有者的用户名
- `%w`：文件创建时间（人类可读），若未知则显示 `-`
- `%W`：文件创建时间（自纪元以来的秒数），若未知则显示 `0`
- `%x`：最后访问时间（人类可读）
- `%X`：最后访问时间（自纪元以来的秒数）
- `%y`：最后修改时间（人类可读）
- `%Y`：最后修改时间（自纪元以来的秒数）
- `%z`：最后状态更改时间（人类可读）
- `%Z`：最后状态更改时间（自纪元以来的秒数）

### `--printf=FORMAT`
类似 `--format`，但会解释反斜杠转义序列，如 `\n`（换行）、`\t`（制表符）等。

### `-t, --terse`
以简洁格式显示信息，所有信息在一行内显示，用空格分隔。适合程序解析。

## 使用示例

1. **显示文件的详细信息**
   ```bash
   stat file.txt
   ```

2. **显示文件系统信息**
   ```bash
   stat -f /home
   ```

3. **自定义输出格式**
   ```bash
   stat -c "文件名: %n, 大小: %s 字节, 权限: %A" file.txt
   ```

4. **显示符号链接指向的文件信息**
   ```bash
   stat -L symbolic_link
   ```

5. **以简洁格式显示**
   ```bash
   stat -t file.txt
   ```

6. **显示文件的所有时间信息**
   ```bash
   stat -c "访问: %x\n修改: %y\n状态更改: %z" file.txt
   ```

7. **显示文件系统类型和块大小**
   ```bash
   stat -f -c "类型: %T\n块大小: %S" /home
   ```

8. **显示文件的 inode 信息**
   ```bash
   stat -c "inode: %i, 链接数: %h" file.txt
   ```

9. **显示文件的所有者信息**
   ```bash
   stat -c "所有者: %U (%u), 组: %G (%g)" file.txt
   ```

10. **显示多个文件的信息**
    ```bash
    stat file1.txt file2.txt file3.txt
    ```

11. **使用 printf 格式并包含换行**
    ```bash
    stat --printf="文件: %n\n大小: %s\n" file.txt
    ```

12. **显示文件创建时间**
    ```bash
    stat -c "创建时间: %w" file.txt
    ```
