# head 命令

## 命令概述

`head` 命令用于输出文件的开头部分。默认情况下，它显示每个文件的前 10 行。可以指定行数、字节数或块数来控制输出量。

**命令格式：**
```
head [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-c` | `--bytes=[-]NUM` | 显示每个文件的前 NUM 个字节。如果 NUM 前有 `-`，则显示除最后 NUM 个字节外的所有内容 |
| `-n` | `--lines=[-]NUM` | 显示每个文件的前 NUM 行。如果 NUM 前有 `-`，则显示除最后 NUM 行外的所有内容 |
| `-q` | `--quiet` | 不输出文件名头信息 |
| | `--silent` | 不输出文件名头信息 |
| `-v` | `--verbose` | 总是输出文件名头信息 |
| `-z` | `--zero-terminated` | 使用 NUL 字符作为行终止符，而不是换行符 |
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `-c, --bytes=[-]NUM`
显示每个文件的前 NUM 个字节。NUM 可以是一个整数，也可以是以下单位之一：
- `b`：512 字节块
- `kB`：1000 字节
- `K`：1024 字节
- `MB`：1000000 字节
- `M`：1048576 字节
- `GB`：1000000000 字节
- `G`：1073741824 字节

如果 NUM 前有 `-`，则显示除最后 NUM 个字节外的所有内容。

### `-n, --lines=[-]NUM`
显示每个文件的前 NUM 行。默认值为 10。如果 NUM 前有 `-`，则显示除最后 NUM 行外的所有内容。

### `-q, --quiet, --silent`
安静模式，不输出文件名头信息。这是处理单个文件时的默认行为。

### `-v, --verbose`
详细模式，总是输出文件名头信息。这是处理多个文件时的默认行为。

### `-z, --zero-terminated`
使用 NUL 字符（\0）作为行终止符，而不是换行符（\n）。这对于处理包含换行符的文件名很有用。

## 使用示例

1. **显示文件的前 10 行（默认）**
   ```bash
   head file.txt
   ```

2. **显示文件的前 20 行**
   ```bash
   head -n 20 file.txt
   head -20 file.txt
   ```

3. **显示文件的前 100 个字节**
   ```bash
   head -c 100 file.txt
   ```

4. **显示文件除最后 5 行外的所有内容**
   ```bash
   head -n -5 file.txt
   ```

5. **显示多个文件的前 5 行，带文件名头**
   ```bash
   head -n 5 -v file1.txt file2.txt
   ```

6. **使用 NUL 作为行终止符**
   ```bash
   head -z -n 5 file.txt
   ```

7. **显示文件的前 2 个块（每块 512 字节）**
   ```bash
   head -c 2b file.txt
   ```

8. **显示文件的前 1KB**
   ```bash
   head -c 1K file.txt
   ```
