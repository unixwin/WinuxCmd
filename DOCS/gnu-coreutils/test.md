# test 命令

## 命令概述

`test` 命令用于检查文件类型并比较值。它评估条件表达式，如果条件为真则返回 0（成功），否则返回非零值（失败）。当以 `[` 命令调用时，需要在条件表达式后添加 `]`。

**命令格式：**
```
test [表达式]
test [选项]
[ 表达式 ]
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

### 文件测试运算符

| 运算符 | 说明 |
|--------|------|
| `-b 文件` | 如果文件存在且是块设备，则为真 |
| `-c 文件` | 如果文件存在且是字符设备，则为真 |
| `-d 文件` | 如果文件存在且是目录，则为真 |
| `-e 文件` | 如果文件存在，则为真 |
| `-f 文件` | 如果文件存在且是普通文件，则为真 |
| `-g 文件` | 如果文件存在且设置了 set-group-id 位，则为真 |
| `-G 文件` | 如果文件存在且属于当前用户的有效组，则为真 |
| `-h 文件` | 如果文件存在且是符号链接，则为真（同 `-L`） |
| `-k 文件` | 如果文件存在且设置了 sticky 位，则为真 |
| `-L 文件` | 如果文件存在且是符号链接，则为真 |
| `-N 文件` | 如果文件存在且自上次读取后已被修改，则为真 |
| `-O 文件` | 如果文件存在且属于当前用户的有效用户，则为真 |
| `-p 文件` | 如果文件存在且是命名管道（FIFO），则为真 |
| `-r 文件` | 如果文件存在且可读，则为真 |
| `-s 文件` | 如果文件存在且大小大于零，则为真 |
| `-S 文件` | 如果文件存在且是套接字，则为真 |
| `-t fd` | 如果文件描述符 fd 在终端上打开，则为真 |
| `-u 文件` | 如果文件存在且设置了 set-user-id 位，则为真 |
| `-w 文件` | 如果文件存在且可写，则为真 |
| `-x 文件` | 如果文件存在且可执行，则为真 |

### 文件比较运算符

| 运算符 | 说明 |
|--------|------|
| `文件1 -nt 文件2` | 如果文件1 比文件2 新（根据修改时间），则为真 |
| `文件1 -ot 文件2` | 如果文件1 比文件2 旧，则为真 |
| `文件1 -ef 文件2` | 如果文件1 和文件2 指向相同的设备和 inode 号，则为真 |

### 字符串运算符

| 运算符 | 说明 |
|--------|------|
| `-z 字符串` | 如果字符串长度为零，则为真 |
| `-n 字符串` | 如果字符串长度非零，则为真 |
| `字符串1 = 字符串2` | 如果字符串相等，则为真 |
| `字符串1 != 字符串2` | 如果字符串不相等，则为真 |
| `字符串1 < 字符串2` | 如果字符串1 在字典序中小于字符串2，则为真 |
| `字符串1 > 字符串2` | 如果字符串1 在字典序中大于字符串2，则为真 |

### 整数比较运算符

| 运算符 | 说明 |
|--------|------|
| `整数1 -eq 整数2` | 如果整数相等，则为真 |
| `整数1 -ne 整数2` | 如果整数不相等，则为真 |
| `整数1 -gt 整数2` | 如果整数1 大于整数2，则为真 |
| `整数1 -ge 整数2` | 如果整数1 大于等于整数2，则为真 |
| `整数1 -lt 整数2` | 如果整数1 小于整数2，则为真 |
| `整数1 -le 整数2` | 如果整数1 小于等于整数2，则为真 |

### 逻辑运算符

| 运算符 | 说明 |
|--------|------|
| `! 表达式` | 逻辑非 |
| `表达式1 -a 表达式2` | 逻辑与 |
| `表达式1 -o 表达式2` | 逻辑或 |
| `( 表达式 )` | 分组（用于改变优先级） |

## 参数详细说明

### `--help`
显示 `test` 命令的使用帮助，包括简要说明和可用选项列表，然后以状态 0 退出。

### `--version`
显示 `test` 命令的版本信息，然后以状态 0 退出。

### 文件测试运算符
用于检查文件的各种属性，如类型、权限、大小等。所有文件测试都跟随符号链接。

### 文件比较运算符
用于比较两个文件的属性，如修改时间、设备号和 inode 号。

### 字符串运算符
用于检查字符串的长度和进行字符串比较。注意 `<` 和 `>` 运算符需要用引号括起来以避免 shell 解释。

### 整数运算符
用于比较两个整数值。整数可以是十进制、八进制（以 0 开头）或十六进制（以 0x 开头）。

### 逻辑运算符
用于组合多个条件。`-a` 的优先级高于 `-o`，但建议使用 shell 的 `&&` 和 `||` 代替。

## 使用示例

1. **检查文件是否存在**
   ```bash
   test -f /etc/passwd && echo "文件存在"
   # 或使用 [ 语法
   [ -f /etc/passwd ] && echo "文件存在"
   ```

2. **检查目录是否存在**
   ```bash
   [ -d /tmp ] && echo "目录存在"
   ```

3. **检查文件是否可读**
   ```bash
   [ -r file.txt ] && echo "文件可读"
   ```

4. **比较字符串**
   ```bash
   [ "$str" = "hello" ] && echo "字符串匹配"
   ```

5. **检查字符串是否为空**
   ```bash
   [ -z "$str" ] && echo "字符串为空"
   [ -n "$str" ] && echo "字符串非空"
   ```

6. **比较整数**
   ```bash
   [ "$a" -gt "$b" ] && echo "a 大于 b"
   [ "$a" -eq "$b" ] && echo "a 等于 b"
   ```

7. **组合条件**
   ```bash
   [ -f file.txt ] && [ -r file.txt ] && echo "文件存在且可读"
   # 或使用 -a 运算符
   [ -f file.txt -a -r file.txt ] && echo "文件存在且可读"
   ```

8. **使用逻辑或**
   ```bash
   [ "$str" = "yes" -o "$str" = "y" ] && echo "用户确认"
   ```

9. **在 if 语句中使用**
   ```bash
   if [ -f "$file" ]; then
       echo "文件存在"
   else
       echo "文件不存在"
   fi
   ```

10. **比较文件修改时间**
    ```bash
    [ file1 -nt file2 ] && echo "file1 比 file2 新"
    ```
