# factor 命令

## 命令概述

`factor` 命令用于打印给定整数的质因数分解。如果未在命令行指定数字，则从标准输入读取（以换行符、制表符或空格分隔）。

**命令格式：**
```
factor [选项]... [数字]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-h` | `--exponents` | 使用指数形式输出（如 `2^3` 而非 `2 2 2`） |

## 参数详细说明

### `-h, --exponents`
以指数形式打印质因数，格式为 `p^e`，而不是重复打印质数 `p` 共 `e` 次。如果指数 `e` 为 1，则省略指数部分。

**默认输出：**
```
$ factor 3000
3000: 2 2 2 3 5 5 5
```

**使用 `--exponents`：**
```
$ factor --exponents 3000
3000: 2^3 3 5^3
```

### 算法说明
- 对于较小的数字（通常小于 2^127），`factor` 使用较快的算法
- 对于更大的数字，使用较慢的算法（Pollard-Brent rho 算法）
- 大质数可以快速识别：分解 Mersenne 质数 2^11213-1 只需约 420 毫秒
- 对于素性测试，`factor` 使用 Baillie-PSW 素性启发式算法

### 输入格式
- 可以从命令行参数或标准输入读取数字
- 多个数字可以用空格、制表符或换行符分隔
- 支持任意精度整数

## 使用示例

1. **分解单个数字**
   ```bash
   factor 60
   # 输出: 60: 2 2 3 5
   ```

2. **分解多个数字**
   ```bash
   factor 12 15 20
   # 输出:
   # 12: 2 2 3
   # 15: 3 5
   # 20: 2 2 5
   ```

3. **使用指数形式**
   ```bash
   factor --exponents 3000
   # 输出: 3000: 2^3 3 5^3
   ```

4. **从标准输入读取**
   ```bash
   echo 1234567890 | factor
   # 输出: 1234567890: 2 3 3 5 3607 3803
   ```

5. **分解大数字**
   ```bash
   factor 4611686018427387903
   ```

6. **检查数字是否为质数**
   ```bash
   factor 17
   # 输出: 17: 17（只有一个因数，说明是质数）
   ```

7. **配合管道使用**
   ```bash
   seq 1 20 | factor
   ```

8. **使用指数形式显示大数分解**
   ```bash
   factor --exponents 1000000
   # 输出: 1000000: 2^6 5^6
   ```
