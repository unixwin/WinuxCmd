# seq 命令

## 命令概述

`seq` 命令用于打印数字序列到标准输出。支持指定起始值、增量和终止值，以及各种输出格式选项。

**命令格式：**
```
seq [选项]... 最后值
seq [选项]... 起始值 最后值
seq [选项]... 起始值 增量 最后值
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-f` | `--format=FORMAT` | 使用 printf 风格的浮点格式字符串 |
| `-s` | `--separator=STRING` | 使用 STRING 分隔数字（默认：换行符） |
| `-w` | `--equal-width` | 通过补零使所有数字等宽 |

## 参数详细说明

### `-f, --format=FORMAT`
使用指定的 printf 风格格式字符串打印所有数字。格式字符串必须恰好包含一个浮点转换规范（`%a`、`%e`、`%f`、`%g`、`%A`、`%E`、`%F`、`%G`）。

`%` 后可跟：
- 零个或多个标志：`-`、`+`、`#`、`0`、` `（空格）、`'`
- 可选宽度：一个或多个数字
- 可选精度：`.` 后跟零个或多个数字
- `%%` 转义表示字面 `%`

**默认格式：** 如果起始值、增量和最后值都使用定点十进制表示，默认格式为 `%.pf`（p 是精确表示输出数字的最小精度）。否则，默认格式为 `%g`。

### `-s, --separator=STRING`
使用 STRING 分隔数字。默认分隔符是换行符。输出始终以换行符结尾。

### `-w, --equal-width`
通过补前导零使所有数字等宽。起始值、增量和最后值应使用定点十进制表示。如需其他类型的填充，请使用 `--format`。

### 参数默认值
- **起始值**：默认为 1
- **增量**：默认为 1（即使起始值大于最后值）
- 如果只指定一个参数，它被视为最后值
- 如果指定两个参数，它们被视为起始值和最后值
- 增量不能为 0

### 特殊值
- 支持负数和递减区间
- 支持 `inf` 表示无穷大
- 不支持 `NaN`
- 十六进制数只有在带有 `p` 风格指数时才能与 `d` 后缀一起使用

### 序列终止条件
当当前数字加上增量的和大于最后值时，序列结束。例如 `seq 1 10 10` 只输出 `1`。

## 使用示例

1. **打印 1 到 5**
   ```bash
   seq 5
   # 输出:
   # 1
   # 2
   # 3
   # 4
   # 5
   ```

2. **打印 3 到 7**
   ```bash
   seq 3 7
   # 输出:
   # 3
   # 4
   # 5
   # 6
   # 7
   ```

3. **指定增量**
   ```bash
   seq 0 2 10
   # 输出: 0 2 4 6 8 10
   ```

4. **递减序列**
   ```bash
   seq 10 -2 0
   # 输出: 10 8 6 4 2 0
   ```

5. **使用自定义分隔符**
   ```bash
   seq -s ', ' 1 5
   # 输出: 1, 2, 3, 4, 5
   ```

6. **等宽补零**
   ```bash
   seq -w 1 10
   # 输出: 01 02 03 ... 10
   ```

7. **使用 printf 格式**
   ```bash
   seq -f '(%9.2E)' -9e5 1.1e6 1.3e6
   # 输出:
   # (-9.00E+05)
   # ( 2.00E+05)
   # ( 1.30E+06)
   ```

8. **生成浮点数序列**
   ```bash
   seq 0 0.1 1
   ```

9. **配合其他命令使用**
   ```bash
   seq 1 10 | xargs -I {} echo "Item {}"
   ```

10. **生成无限序列**
    ```bash
    seq inf | head -5
    # 输出: 1 2 3 4 5
    ```

11. **生成十六进制输出**
    ```bash
    printf '%x\n' $(seq 1048575 1024 1050623)
    ```

12. **使用字符串分隔符**
    ```bash
    seq -s ' + ' 1 5
    # 输出: 1 + 2 + 3 + 4 + 5
    ```
