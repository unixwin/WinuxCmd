# fold 命令

## 命令概述

`fold` 命令用于将长行文本按照指定的宽度进行折叠（换行）。它从指定的文件或标准输入读取内容，并在达到指定宽度时插入换行符。

**命令格式：**
```
fold [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-b` | `--bytes` | 按字节数计算宽度，而非列数 |
| `-s` | `--spaces` | 在空格处断行 |
| `-w` | `--width=宽度` | 设置每行的最大宽度（默认 80 列） |
| | `--help` | 显示帮助信息 |
| | `--version` | 显示版本信息 |

## 参数详细说明

### `-b, --bytes`
按字节数计算行宽，而不是按显示的列数。这对于包含多字节字符（如中文、UTF-8 字符）的文本很有用，因为它将每个字节都计为一个单位。

### `-s, --spaces`
在空格处断行，而不是在任意字符处断行。这可以使输出更易读，避免在单词中间断开。

### `-w, --width=宽度`
设置每行的最大宽度。如果未指定，默认为 80 列。宽度值可以使用 `=` 或空格分隔选项和值。

### `--help`
显示帮助信息并退出。

### `--version`
显示版本信息并退出。

## 使用示例

1. **基本折叠**
   ```bash
   fold file.txt
   ```
   将文件中的长行折叠为 80 列宽度。

2. **设置特定宽度**
   ```bash
   fold -w 60 file.txt
   ```
   将长行折叠为 60 列宽度。

3. **在空格处断行**
   ```bash
   fold -s file.txt
   ```
   在空格处断行，保持单词完整性。

4. **按字节数计算**
   ```bash
   fold -b file.txt
   ```
   按字节数计算宽度，适用于多字节字符。

5. **组合选项**
   ```bash
   fold -w 72 -s file.txt
   ```
   设置 72 列宽度，并在空格处断行。

6. **从标准输入折叠**
   ```bash
   echo "This is a very long line that needs to be folded to fit within the specified width for better readability." | fold -w 40
   ```
   将输入文本折叠为 40 列宽度。

7. **处理 UTF-8 文本**
   ```bash
   fold -b -w 80 chinese.txt
   ```
   按字节数折叠中文文本，宽度为 80 字节。

8. **保持单词完整**
   ```bash
   fold -s -w 50 document.txt
   ```
   在 50 列宽度处折叠，但保持单词不在中间断开。

9. **管道使用**
   ```bash
   cat long_file.txt | fold -w 100 -s
   ```
   读取长文件并以 100 列宽度折叠，在空格处断行。

10. **与其他命令组合**
    ```bash
    ls -l | fold -w 120
    ```
    将 `ls -l` 的输出折叠为 120 列宽度。
