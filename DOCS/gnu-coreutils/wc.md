# wc 命令

## 命令概述

`wc` 命令用于统计文件中的行数、单词数、字节数或字符数。它可以读取一个或多个文件，并输出统计信息。如果没有指定文件，或者文件名为 `-`，则从标准输入读取。

**命令格式：**
```
wc [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-c` | `--bytes` | 统计字节数 |
| `-m` | `--chars` | 统计字符数 |
| `-w` | `--words` | 统计单词数 |
| `-l` | `--lines` | 统计行数 |
| `-L` | `--max-line-length` | 显示最长行的长度 |
| | `--files0-from=文件` | 从指定文件读取以 NUL 结尾的文件名列表 |
| | `--total=样式` | 控制总行的显示方式 |
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `-c, --bytes`
统计文件中的字节数。在多字节字符集中，字节数可能与字符数不同。

### `-m, --chars`
统计文件中的字符数。在多字节字符集中，字符数可能与字节数不同。此选项依赖于 `LC_CTYPE` 环境变量。

### `-w, --words`
统计文件中的单词数。单词是由空白字符分隔的非零长度字符序列。

### `-l, --lines`
统计文件中的行数。行是由换行符分隔的字符序列。

### `-L, --max-line-length`
显示文件中最长行的长度（以字节为单位）。不计算行尾的换行符。

### `--files0-from=文件`
从指定文件中读取以 NUL 字符结尾的文件名列表，而不是从命令行参数读取。如果文件名为 `-`，则从标准输入读取文件名。

### `--total=样式`
控制总行的显示方式。样式可以是：
- `auto`：仅在多个文件时显示总行（默认）
- `always`：总是显示总行
- `only`：只显示总行，不显示单个文件统计

### `--help`
显示帮助信息并退出。

### `--version`
显示版本信息并退出。

## 使用示例

1. **统计文件的行数、单词数和字节数**
   ```bash
   wc file.txt
   ```

2. **只统计行数**
   ```bash
   wc -l file.txt
   ```

3. **只统计单词数**
   ```bash
   wc -w file.txt
   ```

4. **只统计字节数**
   ```bash
   wc -c file.txt
   ```

5. **只统计字符数**
   ```bash
   wc -m file.txt
   ```

6. **显示最长行的长度**
   ```bash
   wc -L file.txt
   ```

7. **统计多个文件**
   ```bash
   wc file1.txt file2.txt file3.txt
   ```

8. **从标准输入统计**
   ```bash
   echo "Hello World" | wc
   ```

9. **统计当前目录下的文件数量**
   ```bash
   ls | wc -l
   ```

10. **使用通配符统计多个文件**
    ```bash
    wc -l *.txt
    ```
