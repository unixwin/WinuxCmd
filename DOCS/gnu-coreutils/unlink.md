# unlink 命令

## 命令概述

`unlink` 命令用于删除单个文件。它调用 `unlink()` 系统调用，删除指定的文件。

**命令格式：**
```
unlink FILE
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `FILE`
要删除的文件路径。`unlink` 只能删除单个文件，不能删除目录。

### `--help`
显示帮助信息，包括命令用法和选项说明，然后退出。

### `--version`
显示版本信息，包括版权和许可证信息，然后退出。

## 使用示例

1. **删除单个文件**
   ```bash
   unlink file.txt
   ```

2. **删除符号链接**
   ```bash
   unlink symlink
   ```

3. **查看帮助信息**
   ```bash
   unlink --help
   ```

4. **查看版本信息**
   ```bash
   unlink --version
   ```
