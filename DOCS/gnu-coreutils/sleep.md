# sleep 命令

## 命令概述

`sleep` 命令用于暂停执行指定的时间。时间由命令行参数的总和决定。

**命令格式：**
```
sleep 数字[后缀]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

> **注意：** GNU `sleep` 没有常规选项。它接受一个或多个时间参数，每个参数是一个非负数后跟可选的单位后缀。

## 参数详细说明

### 时间单位后缀

| 后缀 | 单位 |
|------|------|
| `s` | 秒（默认） |
| `m` | 分钟 |
| `h` | 小时 |
| `d` | 天 |

### 数字格式
- 支持整数和浮点数
- 支持科学计数法（如 `1234e-3` 表示 1.234 秒）
- 支持十六进制数（仅当带有 `p` 风格指数时可与 `d` 后缀一起使用，如 `0x1p0d`）
- 支持 `inf` 表示无限期睡眠
- 可以指定多个参数，它们的值会被累加

### POSIX 兼容性
可移植的 POSIX 脚本只能给 `sleep` 一个不带后缀的非负整数参数。GNU `sleep` 扩展支持多个参数、单位后缀和浮点数。

## 使用示例

1. **睡眠 5 秒**
   ```bash
   sleep 5
   ```

2. **使用浮点数**
   ```bash
   sleep 0.5
   ```

3. **使用分钟为单位**
   ```bash
   sleep 2m
   ```

4. **使用小时为单位**
   ```bash
   sleep 1h
   ```

5. **使用天为单位**
   ```bash
   sleep 1d
   ```

6. **组合多个时间参数**
   ```bash
   sleep 1m 30s
   # 睡眠 90 秒
   ```

7. **使用科学计数法**
   ```bash
   sleep 1234e-3
   # 睡眠 1.234 秒
   ```

8. **无限期睡眠**
   ```bash
   sleep inf
   ```

9. **亚秒级睡眠**
   ```bash
   sleep 0.1
   # 睡眠 100 毫秒
   ```

10. **在脚本中添加延迟**
    ```bash
    for i in $(seq 1 10); do
        echo "Step $i"
        sleep 1
    done
    ```

11. **等待后台进程完成**
    ```bash
    long_task &
    sleep 5
    echo "5 seconds have passed"
    ```
