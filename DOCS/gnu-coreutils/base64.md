# base64 命令

## 命令概述

`base64` 命令用于将数据转换为 base64 编码形式或从 base64 编码形式转换回来。base64 编码形式使用可打印的 ASCII 字符来表示二进制数据。

**命令格式：**
```
base64 [选项]... [文件]
base64 --decode [选项]... [文件]
```

base64 编码会将数据扩展到原始大小的约 133%。该格式符合 RFC 4648 标准。

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-w` | `--wrap=列数` | 编码时在指定列数后换行 |
| `-d` | `--decode` | 从编码数据解码 |
| `-i` | `--ignore-garbage` | 解码时忽略无法识别的字节 |

## 参数详细说明

### `-w, --wrap=列数`
在编码时，在指定的列数后换行。这必须是一个正数。默认是在 76 个字符后换行。使用值 0 可以完全禁用换行。

### `-d, --decode`
改变操作模式，从默认的编码数据变为解码数据。输入应为 base64 编码数据，输出将是原始数据。

### `-i, --ignore-garbage`
解码时，总是接受换行符。在解码过程中，忽略无法识别的字节，以允许解码扭曲的数据。

## 使用示例

1. **编码文件内容**
   ```bash
   base64 file.txt
   ```

2. **解码 base64 编码的数据**
   ```bash
   base64 -d encoded.txt
   ```

3. **编码并限制每行宽度**
   ```bash
   base64 -w 40 file.txt
   ```

4. **从标准输入编码**
   ```bash
   echo "Hello World" | base64
   ```

5. **解码时忽略无效字符**
   ```bash
   base64 -d -i encoded.txt
   ```

6. **编码并完全禁用换行**
   ```bash
   base64 -w 0 file.txt
   ```

7. **编码二进制文件**
   ```bash
   base64 image.png > image.txt
   ```

8. **解码并还原二进制文件**
   ```bash
   base64 -d image.txt > image.png
   ```
