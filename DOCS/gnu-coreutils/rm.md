# rm 命令

## 命令概述

`rm` 命令用于删除文件或目录。默认情况下，`rm` 不删除目录，需要使用 `-r` 选项递归删除目录及其内容。

**命令格式：**
```
rm [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-f` | `--force` | 忽略不存在的文件和参数，从不提示 |
| `-i` | | 每次删除前提示确认 |
| `-I` | | 删除超过三个文件或递归删除时，提示一次确认 |
| `-d` | `--dir` | 删除空目录 |
| `-r` | `--recursive` | 递归删除目录及其内容 |
| `-R` | `--recursive` | 递归删除目录及其内容（同 `-r`） |
| `-v` | `--verbose` | 显示详细操作信息 |
| | `--interactive[=WHEN]` | 根据 WHEN 参数提示：`never`、`once`(`-I`) 或 `always`(`-i`) |
| | `--one-file-system` | 递归删除时，跳过与命令行参数不在同一文件系统的目录 |
| | `--no-preserve-root` | 不对 `/` 特殊处理 |
| | `--preserve-root` | 不删除 `/`（默认行为） |
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `-f, --force`
- 忽略不存在的文件，不报错
- 覆盖任何之前的 `-i` 选项
- 不提示用户确认

### `-i`
在删除每个文件之前提示用户确认。如果有任何否定回答，跳过该文件。

### `-I`
在以下情况提示一次确认：
- 删除超过三个文件
- 递归删除目录

提示内容包含将要执行的操作摘要，用户可以中止整个操作。

### `-d, --dir`
删除空目录。默认情况下 `rm` 不删除目录。

### `-r, --recursive`
递归删除目录及其所有内容。删除过程中会先进入子目录删除其内容，再删除目录本身。

### `-v, --verbose`
显示正在删除的每个文件或目录的名称。

### `--interactive[=WHEN]`
控制交互提示行为：
- `never`：从不提示（等同于 `-f`）
- `once`：提示一次（等同于 `-I`）
- `always`：每次删除前提示（等同于 `-i`）

### `--one-file-system`
在递归删除时，如果遇到与命令行参数不在同一文件系统的目录，跳过该目录并显示警告信息。

### `--no-preserve-root`
不对 `/` 路径进行特殊处理。默认情况下，`rm` 会拒绝递归删除根目录。

### `--preserve-root`
不删除 `/`。这是默认行为。

## 使用示例

1. **删除单个文件**
   ```bash
   rm file.txt
   ```

2. **删除多个文件**
   ```bash
   rm file1.txt file2.txt file3.txt
   ```

3. **递归删除目录**
   ```bash
   rm -r directory/
   ```

4. **删除前提示确认**
   ```bash
   rm -i file.txt
   ```

5. **强制删除（不提示）**
   ```bash
   rm -f file.txt
   ```

6. **删除空目录**
   ```bash
   rm -d empty_dir/
   ```

7. **批量删除前提示一次**
   ```bash
   rm -I *.log
   ```

8. **显示删除过程**
   ```bash
   rm -v file1.txt file2.txt
   ```

9. **递归强制删除**
   ```bash
   rm -rf directory/
   ```

10. **删除前确认每个文件**
    ```bash
    rm -ri old_files/
    ```
