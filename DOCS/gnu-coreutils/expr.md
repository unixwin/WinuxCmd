# expr 命令

## 命令概述

`expr` 命令用于计算表达式的值并将结果输出到标准输出。它支持整数运算、字符串操作和模式匹配。表达式中的操作符需要用空格分隔。

**命令格式：**
```
expr 表达式
expr 选项
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

### 算术运算符

| 运算符 | 说明 |
|--------|------|
| `+` | 加法 |
| `-` | 减法 |
| `*` | 乘法 |
| `/` | 整数除法 |
| `%` | 取模（余数） |

### 字符串运算符

| 运算符 | 说明 |
|--------|------|
| `字符串 : 正则表达式` | 模式匹配，返回匹配的部分 |
| `match 字符串 正则表达式` | 同 `:` 运算符 |
| `substr 字符串 位置 长度` | 提取子字符串 |
| `index 字符串 字符` | 查找字符首次出现的位置 |
| `length 字符串` | 返回字符串长度 |
| `+ TOKEN` | 将 TOKEN 解释为字符串（即使它是运算符或数字） |

### 比较运算符

| 运算符 | 说明 |
|--------|------|
| `=` | 等于 |
| `!=` | 不等于 |
| `<` | 小于 |
| `<=` | 小于等于 |
| `>` | 大于 |
| `>=` | 大于等于 |

### 逻辑运算符

| 运算符 | 说明 |
|--------|------|
| `&` | 逻辑与（如果两个操作数都非零且非空，则返回第一个操作数；否则返回 0） |
| `\|` | 逻辑或（如果第一个操作数非零且非空，则返回它；否则返回第二个操作数） |

## 参数详细说明

### `--help`
显示 `expr` 命令的使用帮助，包括简要说明和可用选项列表，然后以状态 0 退出。

### `--version`
显示 `expr` 命令的版本信息，然后以状态 0 退出。

### 算术运算符
对整数进行基本的算术运算。如果操作数不是整数，`expr` 会报错。注意 `*` 需要用 `\*` 转义或引号括起来以避免 shell 解释为通配符。

### 字符串运算符
`:` 运算符和 `match` 用于正则表达式匹配。正则表达式默认锚定在字符串开头（隐含 `^`）。如果正则表达式中有 `\(` 和 `\)`，则返回第一个括号内的匹配内容。

### 比较运算符
比较结果为真时返回 1，为假时返回 0。注意 `<` 和 `>` 需要转义以避免 shell 解释。

### 逻辑运算符
`&` 运算符在两个操作数都非零且非空时返回第一个操作数，否则返回 0。`|` 运算符在第一个操作数非零且非空时返回它，否则返回第二个操作数。这些运算符短路求值。

## 使用示例

1. **基本算术运算**
   ```bash
   expr 2 + 3        # 输出: 5
   expr 10 - 4       # 输出: 6
   expr 3 \* 4       # 输出: 12
   expr 10 / 3       # 输出: 3
   expr 10 % 3       # 输出: 1
   ```

2. **字符串长度**
   ```bash
   expr length "hello"  # 输出: 5
   ```

3. **提取子字符串**
   ```bash
   expr substr "hello world" 1 5  # 输出: hello
   ```

4. **查找字符位置**
   ```bash
   expr index "hello" "l"    # 输出: 3
   ```

5. **模式匹配**
   ```bash
   expr "hello" : "h\(.*\)o"  # 输出: ell
   expr "hello" : ".*\(l\)"   # 输出: l
   ```

6. **使用 match 运算符**
   ```bash
   expr match "hello" "h\(.*\)o"  # 输出: ell
   ```

7. **比较运算**
   ```bash
   expr 5 \> 3     # 输出: 1
   expr 5 \< 3     # 输出: 0
   expr 5 = 5      # 输出: 1
   ```

8. **逻辑运算**
   ```bash
   expr 1 \& 2     # 输出: 1
   expr 0 \& 2     # 输出: 0
   expr 1 \| 2     # 输出: 1
   expr 0 \| 2     # 输出: 2
   ```

9. **在变量赋值中使用**
   ```bash
   result=$(expr 3 + 4)
   echo $result     # 输出: 7
   ```

10. **在条件语句中使用**
    ```bash
    if expr "$answer" : "yes" > /dev/null; then
        echo "用户确认"
    fi
    ```

11. **递增变量**
    ```bash
    i=1
    i=$(expr $i + 1)
    echo $i  # 输出: 2
    ```
