# date 命令

## 命令概述

`date` 命令用于打印或设置系统日期和时间。当给定格式字符串时，按照指定格式输出当前时间；也可以显示由字符串描述的时间，而非当前时间。

**命令格式：**
```
date [选项]... [+格式]
date [-u|--utc|--universal] [MMDDhhmm[[CC]YY][.ss]]
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-d` | `--date=STRING` | 显示由 STRING 描述的时间，而非 'now' |
| `-f` | `--file=DATEFILE` | 类似 --date，但每次从 DATEFILE 中一行一行读取 |
| `-I[TIMESPEC]` | `--iso-8601[=TIMESPEC]` | 输出 ISO 8601 格式的日期/时间 |
| `-R` | `--rfc-email` | 输出 RFC 5322 兼容的日期字符串 |
| | `--rfc-2822` | 输出 RFC 2822 兼容的日期字符串 |
| | `--rfc-3339=TIMESPEC` | 输出 RFC 3339 兼容的日期/时间 |
| `-r` | `--reference=FILE` | 显示文件的最后修改时间 |
| `-s` | `--set=STRING` | 根据 STRING 设置时间 |
| `-u` | `--utc` | 协调世界时 (UTC) |
| | `--universal` | 同 --utc |
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 输出版本信息并退出 |

## 参数详细说明

### `-d, --date=STRING`
显示由 STRING 描述的时间，而不是当前时间。STRING 可以是各种格式，例如：
- `"2024-01-15 14:30:00"`
- `"last Friday"`
- `"2 days ago"`
- `"@1705312200"` (Unix 时间戳)

### `-f, --file=DATEFILE`
类似 `--date`，但从 DATEFILE 文件中逐行读取日期字符串。如果 DATEFILE 为 `-`，则从标准输入读取。

### `-I[TIMESPEC], --iso-8601[=TIMESPEC]`
以 ISO 8601 格式输出日期和/或时间。TIMESPEC 可选值：
- `date` - 仅输出日期（默认）
- `hours` - 输出日期和小时
- `minutes` - 输出日期和小时:分钟
- `seconds` - 输出日期和小时:分钟:秒
- `ns` / `nanoseconds` - 输出纳秒精度

### `-R, --rfc-email`
输出符合 RFC 5322（电子邮件）格式的日期字符串。

### `--rfc-2822`
输出符合 RFC 2822 格式的日期字符串。

### `--rfc-3339=TIMESPEC`
输出符合 RFC 3339 格式的日期/时间。TIMESPEC 可选值：
- `date` - 输出日期
- `seconds` - 输出日期和秒
- `ns` / `nanoseconds` - 输出纳秒精度

### `-r, --reference=FILE`
显示指定文件的最后修改时间，而非当前时间。

### `-s, --set=STRING`
根据 STRING 设置系统时间（需要适当权限）。

### `-u, --utc, --universal`
使用协调世界时 (UTC) 显示或设置时间。

## 格式化序列

使用 `+FORMAT` 控制输出格式，支持以下转义序列：

| 序列 | 说明 | 示例 |
|------|------|------|
| `%Y` | 年份（4位） | 2024 |
| `%y` | 年份（2位） | 24 |
| `%C` | 世纪 | 20 |
| `%m` | 月份（01-12） | 01 |
| `%d` | 日（01-31） | 15 |
| `%e` | 日（空格填充，1-31） | 15 |
| `%H` | 小时（00-23） | 14 |
| `%I` | 小时（01-12） | 02 |
| `%M` | 分钟（00-59） | 30 |
| `%S` | 秒（00-60） | 45 |
| `%N` | 纳秒 | 000000000 |
| `%p` | AM/PM | PM |
| `%P` | am/pm | pm |
| `%a` | 星期缩写 | Mon |
| `%A` | 星期全称 | Monday |
| `%b` | 月份缩写 | Jan |
| `%B` | 月份全称 | January |
| `%j` | 年内天数（001-366） | 015 |
| `%u` | 星期几（1-7，1=周一） | 1 |
| `%w` | 星期几（0-6，0=周日） | 1 |
| `%s` | Unix 时间戳 | 1705312200 |
| `%z` | 时区偏移 | +0800 |
| `%:z` | 时区偏移（带冒号） | +08:00 |
| `%Z` | 时区名称 | CST |
| `%F` | 完整日期（等同 %Y-%m-%d） | 2024-01-15 |
| `%D` | 日期（等同 %m/%d/%y） | 01/15/24 |
| `%T` | 时间（等同 %H:%M:%S） | 14:30:45 |
| `%R` | 时间（等同 %H:%M） | 14:30 |
| `%r` | 12小时制时间 | 02:30:45 PM |
| `%c` | 本地日期时间 | Mon Jan 15 14:30:45 2024 |
| `%x` | 本地日期 | 01/15/24 |
| `%X` | 本地时间 | 14:30:45 |
| `%n` | 换行符 | |
| `%t` | 制表符 | |
| `%%` | 百分号 | % |

## 使用示例

1. **显示当前日期和时间**
   ```bash
   date
   ```

2. **显示指定格式的日期**
   ```bash
   date +'%Y-%m-%d %H:%M:%S'
   ```

3. **显示 UTC 时间**
   ```bash
   date -u
   ```

4. **显示 RFC 2822 格式的日期**
   ```bash
   date -R
   ```

5. **显示 ISO 8601 格式的日期**
   ```bash
   date -I
   ```

6. **显示 ISO 8601 格式的日期和秒**
   ```bash
   date -Iseconds
   ```

7. **显示指定日期字符串的时间**
   ```bash
   date -d '2024-01-15 14:30:00'
   ```

8. **显示文件的最后修改时间**
   ```bash
   date -r /etc/passwd
   ```

9. **显示 Unix 时间戳**
   ```bash
   date +%s
   ```

10. **将时间戳转换为日期**
    ```bash
    date -d @1705312200
    ```

11. **显示昨天的日期**
    ```bash
    date -d 'yesterday' +'%Y-%m-%d'
    ```

12. **显示下周一的日期**
    ```bash
    date -d 'next Monday' +'%Y-%m-%d'
    ```
