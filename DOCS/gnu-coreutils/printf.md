# printf 命令

## 命令概述

`printf` 命令用于格式化输出文本。它类似于 C 语言中的 `printf` 函数，提供了更强大的格式化功能，包括格式说明符、转义字符和宽度控制。

**命令格式：**
```
printf 格式 [参数]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `--help` | | 显示帮助信息并退出 |
| `--version` | | 显示版本信息并退出 |

## 参数详细说明

### 格式字符串
格式字符串包含普通字符和格式说明符。普通字符按原样输出，格式说明符用于格式化参数。

### 格式说明符
格式说明符以 `%` 开头，后跟可选的修饰符和转换字符：
- `%s`：字符串
- `%d` 或 `%i`：十进制整数
- `%u`：无符号十进制整数
- `%o`：八进制整数
- `%x` 或 `%X`：十六进制整数
- `%f` 或 `%F`：浮点数
- `%e` 或 `%E`：科学计数法浮点数
- `%g` 或 `%G`：自动选择 `%f` 或 `%e`
- `%c`：字符
- `%b`：转义字符串（类似 `%s`，但解释转义序列）
- `%q`：带引号的字符串（安全用于 shell 重新输入）
- `%%`：输出字面量 `%`

### 修饰符
- `标志`：
  - `-`：左对齐
  - `+`：总是显示符号
  - ` `：正数前加空格
  - `0`：用零填充
  - `#`：使用替代形式
- `宽度`：最小字段宽度
- `精度`：小数点后的位数（对于浮点数）或最大字符串长度
- `长度修饰符`：
  - `hh`：char
  - `h`：short
  - `l`：long
  - `ll`：long long
  - `L`：long double

### 转义序列
`printf` 支持以下转义序列：
- `\a`：警报（BEL）
- `\b`：退格
- `\c`：不产生进一步输出
- `\e`：转义字符
- `\f`：换页
- `\n`：换行
- `\r`：回车
- `\t`：水平制表符
- `\v`：垂直制表符
- `\\`：反斜杠
- `\0NNN`：八进制值 NNN 表示的字节
- `\xHH`：十六进制值 HH 表示的字节
- `\uHHHH`：Unicode 字符（4 位十六进制）
- `\UHHHHHHHH`：Unicode 字符（8 位十六进制）

## 使用示例

1. **输出简单文本**
   ```bash
   printf "Hello, World!\n"
   ```

2. **格式化输出字符串**
   ```bash
   printf "姓名: %s, 年龄: %d\n" "张三" 25
   ```

3. **控制输出宽度**
   ```bash
   printf "%-20s %10d\n" "产品" "价格"
   ```

4. **输出浮点数**
   ```bash
   printf "圆周率: %.2f\n" 3.14159
   ```

5. **输出十六进制数**
   ```bash
   printf "十六进制: %x\n" 255
   ```

6. **输出八进制数**
   ```bash
   printf "八进制: %o\n" 255
   ```

7. **输出带前导零的数字**
   ```bash
   printf "编号: %05d\n" 42
   ```

8. **输出科学计数法**
   ```bash
   printf "科学计数法: %e\n" 123456.789
   ```

9. **输出带引号的字符串**
   ```bash
   printf "安全字符串: %q\n" "Hello World"
   ```

10. **输出字符**
    ```bash
    printf "字符: %c\n" 65
    ```

11. **输出百分号**
    ```bash
    printf "完成: 100%%\n"
    ```

12. **左对齐输出**
    ```bash
    printf "%-10s %-10s\n" "左对齐" "右对齐"
    ```

13. **使用宽度和精度**
    ```bash
    printf "宽度10，精度2: %10.2f\n" 3.14159
    ```

14. **输出多个参数**
    ```bash
    printf "%s %s %s\n" "一" "二" "三"
    ```

15. **输出转义字符**
    ```bash
    printf "制表符:\t换行:\n"
    ```

16. **输出 Unicode 字符**
    ```bash
    printf "Unicode: \u4F60\u597D\n"
    ```
