# who - 显示当前登录的用户信息

## 命令概述

`who` 命令用于显示当前登录系统的用户信息，包括用户名、终端、登录时间等详细信息。这是一个用于系统监控和管理的实用工具，比 `users` 命令提供更详细的信息。

## 完整参数列表

| 短选项 | 长选项 | 描述 |
|--------|--------|------|
| `-a` | `--all` | 显示所有信息（等同于 `-b -d --login -p -r -t -T -u`） |
| `-b` | `--boot` | 显示上次系统启动时间 |
| `-d` | `--dead` | 显示已终止的进程 |
| `-H` | `--heading` | 显示标题行 |
| `-l` | `--login` | 显示系统登录进程 |
| | `--lookup` | 尝试规范化主机名通过 DNS |
| `-m` | | 仅显示与当前终端关联的主机名和用户 |
| `-p` | `--process` | 显示由 init 生成的活动进程 |
| `-q` | `--count` | 显示所有登录用户名和数量 |
| `-r` | `--runlevel` | 显示当前运行级别 |
| `-s` | | 仅显示名称、行和时间字段（默认） |
| `-t` | `--time` | 显示上次系统时钟更改 |
| `-T` | `-w` `--mesg` | 显示用户的消息状态（`+`、`-` 或 `?`） |
| `-u` | `--users` | 显示已登录的用户 |
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `-a, --all`
显示所有可用信息。这等同于同时使用 `-b -d --login -p -r -t -T -u` 选项。

### `-b, --boot`
显示上次系统启动的时间和日期。

### `-d, --dead`
显示已终止的进程。这有助于识别异常退出的会话。

### `-H, --heading`
在输出顶部显示标题行，包括列名。

### `-l, --login`
显示系统登录进程。

### `--lookup`
尝试通过 DNS 规范化主机名。这可能会减慢输出速度。

### `-m`
仅显示与当前终端关联的主机名和用户。这与 `who am i` 命令相同。

### `-p, --process`
显示由 init 生成的活动进程。

### `-q, --count`
显示所有登录用户名和登录用户总数。这提供了简洁的登录用户概览。

### `-r, --runlevel`
显示当前系统运行级别。

### `-s`
仅显示名称、行和时间字段。这是默认行为。

### `-t, --time`
显示上次系统时钟更改的时间。

### `-T, -w, --mesg`
显示用户的消息状态：
- `+`：消息已启用
- `-`：消息已禁用
- `?`：无法找到终端设备

### `-u, --users`
显示已登录的用户及其空闲时间和进程 ID。

### `--help`
显示 `who` 命令的帮助信息，包括所有可用选项和简要说明，然后退出。

### `--version`
显示 `who` 命令的版本信息，然后退出。

## 使用示例

### 基本用法
```bash
$ who
john     pts/0        2024-01-15 09:30 (192.168.1.100)
jane     pts/1        2024-01-15 10:15 (192.168.1.101)
```

### 显示所有信息
```bash
$ who -a
           system boot  2024-01-15 08:00
           run-level 5  2024-01-15 08:00
john     + pts/0        2024-01-15 09:30 00:05        1234 (192.168.1.100)
jane     + pts/1        2024-01-15 10:15   .          5678 (192.168.1.101)
```

### 显示标题
```bash
$ who -H
NAME     LINE         TIME             COMMENT
john     pts/0        2024-01-15 09:30 (192.168.1.100)
jane     pts/1        2024-01-15 10:15 (192.168.1.101)
```

### 显示用户消息状态
```bash
$ who -T
john     + pts/0        2024-01-15 09:30 (192.168.1.100)
jane     - pts/1        2024-01-15 10:15 (192.168.1.101)
```

### 显示登录用户数量
```bash
$ who -q
john jane
# users=2
```

### 显示上次系统启动时间
```bash
$ who -b
         system boot  2024-01-15 08:00
```

### 显示当前运行级别
```bash
$ who -r
         run-level 5  2024-01-15 08:00
```

### 在脚本中使用
```bash
#!/bin/bash
# 检查用户是否登录
if who | grep -q "john"; then
    echo "用户 john 已登录"
else
    echo "用户 john 未登录"
fi

# 统计登录用户数量
login_count=$(who | wc -l)
echo "当前有 $login_count 个用户登录"

# 显示系统启动时间
boot_time=$(who -b | awk '{print $3, $4}')
echo "系统启动时间: $boot_time"
```
