# true 命令

## 命令概述

`true` 命令不执行任何操作，仅返回退出状态 0（表示成功）。它通常用于 shell 脚本中作为占位符或在条件语句中强制成功。

**命令格式：**
```
true [忽略的参数]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `--help`
显示 `true` 命令的使用帮助，包括简要说明和可用选项列表，然后以状态 0 退出。

### `--version`
显示 `true` 命令的版本信息，然后以状态 0 退出。

## 使用示例

1. **基本使用 - 返回成功状态**
   ```bash
   true
   echo $?  # 输出: 0
   ```

2. **在条件语句中使用**
   ```bash
   if true; then
       echo "true 命令返回成功"
   else
       echo "这行不会执行"
   fi
   ```

3. **创建无限循环**
   ```bash
   while true; do
       echo "循环中..."
       sleep 1
   done
   ```

4. **忽略参数**
   ```bash
   true --any-argument
   echo $?  # 输出: 0
   ```

5. **在逻辑或操作中使用**
   ```bash
   false || true
   echo $?  # 输出: 0
   ```

6. **在脚本中作为占位符**
   ```bash
   #!/bin/bash
   # TODO: 实现此功能
   true
   ```
