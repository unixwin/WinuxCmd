# cat 命令

## 命令概述

`cat` 命令用于连接文件并打印到标准输出。它可以读取一个或多个文件，并将它们的内容连续输出。如果没有指定文件，或者文件名为 `-`，则从标准输入读取。

**命令格式：**
```
cat [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-A` | `--show-all` | 等价于 `-vET` |
| `-b` | `--number-nonblank` | 对非空输出行编号，从 1 开始 |
| `-e` | | 等价于 `-vE` |
| `-E` | `--show-ends` | 在每行末尾显示 `$` |
| `-n` | `--number` | 对所有输出行编号，从 1 开始 |
| `-s` | `--squeeze-blank` | 压缩连续的空行，只输出一个空行 |
| `-t` | | 等价于 `-vT` |
| `-T` | `--show-tabs` | 将 TAB 字符显示为 `^I` |
| `-u` | | 忽略；用于 POSIX 兼容性 |
| `-v` | `--show-nonprinting` | 使用 `^` 符号显示控制字符（LFD 和 TAB 除外），并用 `M-` 前置高位设置的字符 |

## 参数详细说明

### `-A, --show-all`
等价于 `-vET`，同时显示非打印字符、行尾标记和制表符。

### `-b, --number-nonblank`
对非空行进行编号，空行不编号但计数继续。如果同时使用 `-n` 选项，此选项优先。

### `-E, --show-ends`
在每行末尾添加 `$` 符号，包括最后一行。`\r\n` 组合显示为 `^M$`。

### `-n, --number`
对所有输出行进行编号，包括空行。如果同时使用 `-b` 选项，则此选项被忽略。

### `-s, --squeeze-blank`
将多个连续的空行压缩为单个空行输出。

### `-T, --show-tabs`
将制表符显示为 `^I`，而不是空白。

### `-v, --show-nonprinting`
显示控制字符（LFD 和 TAB 除外），使用 `^` 符号表示（例如，`^I` 表示制表符，`^M` 表示回车）。高位设置的字符（字符值 >= 128）使用 `M-` 前缀表示。

## 使用示例

1. **显示文件内容**
   ```bash
   cat file.txt
   ```

2. **连接多个文件**
   ```bash
   cat file1.txt file2.txt > combined.txt
   ```

3. **显示行号（包括空行）**
   ```bash
   cat -n file.txt
   ```

4. **显示非空行的行号**
   ```bash
   cat -b file.txt
   ```

5. **压缩连续空行并显示行尾标记**
   ```bash
   cat -sE file.txt
   ```

6. **显示所有非打印字符**
   ```bash
   cat -A file.txt
   ```

7. **从标准输入读取并显示**
   ```bash
   echo "Hello World" | cat
   ```

8. **先显示文件内容，再显示标准输入，最后显示另一个文件**
   ```bash
   cat f - g
   ```
