# nl 命令

## 命令概述

`nl` 命令用于计算文件中的行号。它读取每个文件（`-` 表示标准输入），并将内容输出到标准输出，同时在某些或所有行的开头添加行号。

**命令格式：**
```
nl [选项]... [文件]...
```

`nl` 将其输入分解为逻辑页面部分；默认情况下，在每个逻辑页面部分将行号重置为 1。`nl` 将所有输入文件视为单个文档；它不会在文件之间重置行号或逻辑页面。

一个逻辑页面由三个部分组成：页眉、正文和页脚。每个部分都可以为空。每个部分可以使用不同的编号样式。

逻辑页面各部分的开始由输入文件中包含以下分隔符字符串之一的行指示：
- `\:\:\:` - 页眉开始
- `\:\:` - 正文开始
- `\:` - 页脚开始

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-b` | `--body-numbering=样式` | 设置正文部分的编号样式 |
| `-d` | `--section-delimiter=字符` | 设置分隔符字符 |
| `-f` | `--footer-numbering=样式` | 设置页脚部分的编号样式 |
| `-h` | `--header-numbering=样式` | 设置页眉部分的编号样式 |
| `-i` | `--line-increment=数字` | 设置行号增量 |
| `-l` | `--join-blank-lines=数字` | 设置合并空行的数量 |
| `-n` | `--number-format=格式` | 设置行号格式 |
| `-p` | `--no-renumber` | 不在逻辑页面开始时重置行号 |
| `-s` | `--number-separator=字符串` | 设置行号与文本之间的分隔符 |
| `-v` | `--starting-line-number=数字` | 设置起始行号 |
| `-w` | `--number-width=数字` | 设置行号宽度 |

## 参数详细说明

### `-b, --body-numbering=样式`
为逻辑页面正文部分的行选择编号样式。样式可以是：
- `a`：对所有行编号
- `t`：只对非空行编号（正文的默认值）
- `n`：不对行编号（页眉和页脚的默认值）
- `pbre`：只对包含基本正则表达式 `bre` 匹配的行编号

### `-d, --section-delimiter=字符`
设置分隔符字符，默认为 `\:`。如果只给出一个字符，第二个字符保持为 `:`。作为 GNU 扩展，可以指定多个字符。如果字符为空（`-d ''`），则禁用分节匹配。

### `-f, --footer-numbering=样式`
类似于 `--body-numbering`，但用于页脚部分。

### `-h, --header-numbering=样式`
类似于 `--body-numbering`，但用于页眉部分。

### `-i, --line-increment=数字`
设置行号增量，默认为 1。数字可以是负数以递减。

### `-l, --join-blank-lines=数字`
将连续的 `数字` 个空行视为一个逻辑行进行编号，并且只对最后一行编号。如果连续空行少于 `数字`，则不对它们编号。空行是指不包含任何字符（甚至没有空格或制表符）的行。

### `-n, --number-format=格式`
选择行号格式（默认为 `rn`）：
- `ln`：左对齐，无前导零
- `rn`：右对齐，无前导零
- `rz`：右对齐，有前导零

### `-p, --no-renumber`
不在逻辑页面开始时重置行号。

### `-s, --number-separator=字符串`
在输出中，使用指定的字符串分隔行号和文本行（默认为制表符）。

### `-v, --starting-line-number=数字`
设置每个逻辑页面的起始行号（默认为 1）。起始数字可以是负数。

### `-w, --number-width=数字`
使用指定数量的字符显示行号（默认为 6）。

## 使用示例

1. **对文件添加行号**
   ```bash
   nl file.txt
   ```

2. **对所有行编号（包括空行）**
   ```bash
   nl -ba file.txt
   ```

3. **使用三位数宽度显示行号**
   ```bash
   nl -w 3 file.txt
   ```

4. **使用前导零显示行号**
   ```bash
   nl -n rz file.txt
   ```

5. **设置起始行号为 10**
   ```bash
   nl -v 10 file.txt
   ```

6. **使用自定义分隔符**
   ```bash
   nl -s ". " file.txt
   ```

7. **只对非空行编号（默认行为）**
   ```bash
   nl -b t file.txt
   ```

8. **每两行空行合并编号**
   ```bash
   nl -l 2 file.txt
   ```
