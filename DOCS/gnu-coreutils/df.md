# df 命令

## 命令概述

`df` 命令用于显示文件系统的磁盘空间使用情况。它报告已挂载文件系统的总空间、已用空间、可用空间以及使用百分比等信息。

**命令格式：**
```
df [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-a` | `--all` | 包含所有文件系统，包括大小为 0 的块 |
| `-B` | `--block-size=SIZE` | 按指定大小缩放尺寸输出 |
| `-h` | `--human-readable` | 以人类可读格式显示（例如：1K, 234M, 2G） |
| `-H` | `--si` | 以 1000 为基数而非 1024 |
| `-i` | `--inodes` | 显示 inode 信息而非块使用情况 |
| `-k` | | 类似 `--block-size=1K` |
| `-l` | `--local` | 仅显示本地文件系统 |
| `--no-sync` | | 在获取使用信息前不调用同步（默认） |
| `--output[=FIELD_LIST]` | | 使用字段列表定义输出格式 |
| `-P` | `--portability` | 使用 POSIX 输出格式 |
| `--sync` | | 在获取使用信息前调用同步 |
| `--total` | | 显示所有文件系统的总计行 |
| `-t` | `--type=TYPE` | 仅显示指定类型的文件系统 |
| `-T` | `--print-type` | 显示文件系统类型 |
| `-x` | `--exclude-type=TYPE` | 排除指定类型的文件系统 |
| `--help` | | 显示帮助信息并退出 |
| `--version` | | 显示版本信息并退出 |

## 参数详细说明

### `-a, --all`
包含所有文件系统，包括大小为 0 的块文件系统、伪文件系统和重复的文件系统。默认情况下，这些文件系统会被忽略。

### `-B, --block-size=SIZE`
按指定大小缩放尺寸输出。SIZE 可以是以下值：
- `KB` = 1000
- `K` = 1024
- `MB` = 1000*1000
- `M` = 1024*1024
- 等等，直到 `EB`、`E`

### `-h, --human-readable`
以人类可读格式显示大小，自动选择最合适的单位（K、M、G、T、P、E）。

### `-H, --si`
类似 `-h`，但使用 1000 为基数而非 1024。

### `-i, --inodes`
显示 inode 使用信息而非块使用情况。显示每个文件系统的 inode 总数、已用数和可用数。

### `-k`
等价于 `--block-size=1K`，以 1024 字节为单位显示。

### `-l, --local`
仅显示本地文件系统，排除网络文件系统等远程文件系统。

### `--no-sync`
在获取磁盘使用信息前不调用 sync 系统调用。这是默认行为。

### `--output[=FIELD_LIST]`
使用自定义字段列表定义输出格式。可用字段包括：`source`、`fstype`、`itotal`、`iused`、`iavail`、`ipcent`、`size`、`used`、`avail`、`pcent`、`file`、`target`。

### `-P, --portability`
使用 POSIX 输出格式。每行输出一行信息，包含文件系统、总块数、已用块数、可用块数、使用百分比和挂载点。

### `--sync`
在获取磁盘使用信息前调用 sync 系统调用，确保数据一致性。

### `--total`
在输出末尾添加一行总计信息，显示所有文件系统的总空间、已用空间和可用空间。

### `-t, --type=TYPE`
仅显示指定类型的文件系统。可以多次使用此选项指定多种类型。常见类型包括：`ext4`、`xfs`、`nfs`、`tmpfs` 等。

### `-T, --print-type`
在输出中显示文件系统类型列。

### `-x, --exclude-type=TYPE`
排除指定类型的文件系统。可以多次使用此选项排除多种类型。

## 使用示例

1. **显示所有文件系统的磁盘使用情况**
   ```bash
   df
   ```

2. **以人类可读格式显示**
   ```bash
   df -h
   ```

3. **显示 inode 信息**
   ```bash
   df -i
   ```

4. **显示文件系统类型**
   ```bash
   df -T
   ```

5. **仅显示 ext4 类型的文件系统**
   ```bash
   df -t ext4
   ```

6. **排除 tmpfs 文件系统**
   ```bash
   df -x tmpfs
   ```

7. **显示本地文件系统**
   ```bash
   df -l
   ```

8. **以 1000 为基数显示人类可读格式**
   ```bash
   df -H
   ```

9. **显示总计信息**
   ```bash
   df --total
   ```

10. **自定义输出格式**
    ```bash
    df --output=source,fstype,size,used,avail,pcent,target
    ```

11. **显示特定文件的文件系统信息**
    ```bash
    df /home/user/file.txt
    ```
