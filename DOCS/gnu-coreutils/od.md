# od 命令

## 命令概述

`od` 命令用于以八进制或其他格式转储文件内容。它以明确的方式表示每个文件（`-` 表示标准输入）的内容。

**命令格式：**
```
od [选项]... [文件]...
od [-abcdfilosx]... [文件] [[+]偏移量[.][b]]
od [选项]... --traditional [文件] [[+]偏移量[.][b] [[+]标签[.][b]]]
```

每行输出包括输入中的偏移量，后跟来自文件的数据组。默认情况下，`od` 以八进制打印偏移量，每个文件数据组是一个 C `short int` 的数据量，以单个八进制数打印。

## 参数详细说明

### `-A, --address-radix=基数`
选择文件偏移量的打印基数。基数可以是以下之一：
- `d`：十进制
- `o`：八进制（默认）
- `x`：十六进制
- `n`：无（不打印偏移量）

### `--endian=顺序`
重新排序输入字节，以处理具有不同字节顺序的输入，或提供与当前系统字节顺序无关的一致输出。交换根据指定的 `--type` 大小和字节顺序（`little` 或 `big`）进行。

### `-j, --skip-bytes=字节数`
在格式化和写入之前跳过指定数量的输入字节。如果字节数以 `0x` 或 `0X` 开头，则以十六进制解释；否则，如果以 `0` 开头，则以八进制解释；否则，以十进制解释。字节数可以是整数，后跟以下乘法后缀之一：
- `b` => 512（"块"）
- `KB` => 1000（千字节）
- `K` => 1024（千比字节）
- `MB` => 1000*1000（兆字节）
- `M` => 1024*1024（兆比字节）
- `GB` => 1000*1000*1000（吉字节）
- `G` => 1024*1024*1024（吉比字节）
- 以及 `T`、`P`、`E`、`Z`、`Y`、`R` 和 `Q`。

### `-N, --read-bytes=字节数`
最多输出指定数量的字节。前缀和后缀的解释与 `-j` 选项相同。

### `-S, --strings=字节数`
代替正常输出，只输出字符串常量：至少 `字节数` 个连续的可打印字符，后跟一个零字节（ASCII NUL）。如果与 `-N` 选项组合使用，则截断的字符串被视为以 ASCII NUL 终止。如果省略字节数，默认为 3。

### `-t, --format=类型`
选择输出文件数据的格式。类型是一个或多个类型指示字符的字符串。如果包含多个类型指示字符，或多次使用此选项，`od` 将使用您指定的每种数据类型，按指定的顺序，为每行输出写入一个副本。

在任何类型规范后添加尾随的 "z" 会将单字节字符表示附加到该类型规范生成的输出行。

类型指示字符：
- `a`：命名字符，忽略高位
- `c`：可打印单字节字符，C 反斜杠转义或 3 位八进制序列
- `d`：有符号十进制
- `f`：浮点数
- `o`：八进制
- `u`：无符号十进制
- `x`：十六进制

对于整数类型（`d`、`o`、`u`、`x`），可以指定用于解释每个数字的字节数，或使用以下字符指定 C 编译器内置数据类型的大小：
- `C`：char
- `S`：short
- `I`：int
- `L`：long

对于浮点数（`f`）：
- `B`：brain 16 位浮点数
- `H`：半精度浮点数
- `F`：float
- `D`：double
- `L`：long double

### `-v, --output-duplicates`
输出连续的重复行。默认情况下，当两个或多个连续输出行相同时，`od` 只输出第一行，并在后面的行上放一个星号以表示省略。

### `-w, --width=宽度`
每行输出指定数量的输入字节。这必须是指定输出类型大小的最小公倍数的倍数。如果未指定此选项，默认为 16。如果省略宽度值，默认为 32。

## 使用示例

1. **以八进制显示文件内容**
   ```bash
   od file.txt
   ```

2. **以十六进制显示文件内容**
   ```bash
   od -x file.txt
   ```

3. **以字符形式显示文件内容**
   ```bash
   od -c file.txt
   ```

4. **跳过前 10 个字节**
   ```bash
   od -j 10 file.txt
   ```

5. **只读取 20 个字节**
   ```bash
   od -N 20 file.txt
   ```

6. **显示字符串常量**
   ```bash
   od -S 4 file.txt
   ```

7. **以十进制显示**
   ```bash
   od -d file.txt
   ```

8. **使用传统语法**
   ```bash
   od --traditional file.txt +10
   ```
