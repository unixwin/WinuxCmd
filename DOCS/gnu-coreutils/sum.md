# sum 命令

## 命令概述

`sum` 命令用于计算文件的校验和与块计数。它使用 BSD 校验和算法或 System V 校验和算法来验证文件的完整性。

**命令格式：**
```
sum [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-r` | | 使用 BSD 校验和算法（默认） |
| `-s` | | 使用 System V 校验和算法 |
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `-r`
使用 BSD 校验和算法（16 位校验和）。这是默认算法。输出格式为：校验和 块数 文件名。

### `-s`
使用 System V 校验和算法（16 位校验和）。输出格式为：校验和 块数 文件名。

### `--help`
显示帮助信息并退出。

### `--version`
显示版本信息并退出。

## 使用示例

1. **计算文件的 BSD 校验和（默认）**
   ```bash
   sum file.txt
   ```

2. **计算文件的 System V 校验和**
   ```bash
   sum -s file.txt
   ```

3. **计算多个文件的校验和**
   ```bash
   sum file1.txt file2.txt
   ```

4. **从标准输入计算校验和**
   ```bash
   echo "Hello World" | sum
   ```

5. **比较两个文件的校验和**
   ```bash
   sum file1.txt file2.txt
   ```
