# pr 命令

## 命令概述

`pr` 命令用于将文本文件转换为适合打印的格式。它可以为文件添加页眉、页脚、行号、分页等，使其更适合在打印机上输出。

**命令格式：**
```
pr [选项]... [文件]...
```

## 参数详细说明

### `-a, --across`
按列横向排列文件内容。默认情况下，`pr` 按列纵向排列，使用此选项后，内容将从左到右横向填充列。

### `-b, --balance`
平衡最后一列的列数，使各列的行数尽可能相等。

### `-c, --show-control-chars`
使用 `^` 符号显示控制字符，而不是使用八进制表示。

### `-d, --double-space`
使用双倍行距输出，即在每行之间插入一个空行。

### `-e, --expand-tabs=字符数`
将制表符扩展为指定数量的空格。默认为 8 个空格。

### `-f, -F, --form-feed`
使用换页符（form feed）而不是多个换行符来分页。

### `-h, --header=标题`
设置页眉的标题文本。默认为文件名和日期。

### `-i, --output-tabs=字符数`
使用制表符替代空格来缩进。默认为 8 个字符。

### `-J, --join-lines`
合并行，防止被 `-W` 选项截断。这对于长行很有用。

### `-l, --length=行数`
设置每页的行数。默认为 66 行。

### `-m, --merge`
合并多个文件，并排显示。每个文件占据一列。

### `-n, --number-lines=宽度`
为每行添加行号。可以指定行号的宽度，默认为 5 个字符。

### `-N, --first-line-number=行号`
设置起始行号。默认为 1。

### `-o, --indent=空格数`
设置左边距的空格数。默认为 0。

### `-r, --no-file-warnings`
无法打开文件时不显示警告信息。

### `-s, --separator=字符`
设置列之间的分隔字符。默认为空格。

### `-S, --sep-string=字符串`
设置列之间的分隔字符串。默认为空格。

### `-t, --omit-header`
不输出页眉和页脚。这在只需要格式化内容时很有用。

### `-T, --omit-pagination`
不分页，不输出页眉和页脚。这在将输出重定向到文件时很有用。

### `-v, --show-non-printing`
使用 `^` 和 `M-` 前缀显示非打印字符。

### `-w, --width=宽度`
设置每行的宽度。默认为 72 个字符。

### `-W, --page-width=宽度`
设置页面的宽度。默认为 72 个字符。

### `--help`
显示帮助信息并退出。

### `--version`
显示版本信息并退出。

## 使用示例

1. **基本打印格式化**
   ```bash
   pr file.txt
   ```
   将文件转换为标准打印格式，包含页眉和页脚。

2. **设置页面宽度**
   ```bash
   pr -w 80 file.txt
   ```
   设置页面宽度为 80 个字符。

3. **添加行号**
   ```bash
   pr -n file.txt
   ```
   为每行添加行号。

4. **设置起始行号**
   ```bash
   pr -n -N 100 file.txt
   ```
   从第 100 行开始编号。

5. **合并多个文件**
   ```bash
   pr -m file1.txt file2.txt file3.txt
   ```
   将三个文件并排显示。

6. **设置页眉标题**
   ```bash
   pr -h "My Document" file.txt
   ```
   设置页眉标题为 "My Document"。

7. **省略页眉页脚**
   ```bash
   pr -t file.txt
   ```
   只输出格式化的内容，不包含页眉页脚。

8. **设置每页行数**
   ```bash
   pr -l 50 file.txt
   ```
   设置每页为 50 行。

9. **双倍行距**
   ```bash
   pr -d file.txt
   ```
   使用双倍行距输出。

10. **设置列分隔符**
    ```bash
    pr -s "," -m file1.txt file2.txt
    ```
    使用逗号作为列分隔符合并文件。

11. **从标准输入格式化**
    ```bash
    ls -l | pr -t -w 120
    ```
    将 `ls -l` 的输出格式化为 120 列宽度，不包含页眉页脚。
