# tail 命令

## 命令概述

`tail` 命令用于输出文件的末尾部分。默认情况下，它显示每个文件的最后 10 行。可以指定行数、字节数或块数来控制输出量。`tail` 还支持跟踪模式，可以实时监控文件的新增内容。

**命令格式：**
```
tail [选项]... [文件]...
```

## 完整参数列表

| 短选项 | 长选项 | 说明 |
|--------|--------|------|
| `-c` | `--bytes=[+]NUM` | 显示每个文件的最后 NUM 个字节。如果 NUM 前有 `+`，则从第 NUM 个字节开始显示 |
| `-n` | `--lines=[+]NUM` | 显示每个文件的最后 NUM 行。如果 NUM 前有 `+`，则从第 NUM 行开始显示 |
| `-f` | `--follow[=how]` | 跟踪文件末尾的增长。how 可以是 `descriptor`（默认）或 `name` |
| `-F` | `--follow=name --retry` | 跟踪文件名，即使文件被轮转或重新创建 |
| | `--max-unchanged-stats=N` | 使用 `--follow=name` 时，在文件大小不变 N 次后重新打开文件 |
| | `--pid=PID` | 与 `-f` 一起使用，在进程 PID 结束后终止 |
| `-q` | `--quiet` | 不输出文件名头信息 |
| | `--silent` | 不输出文件名头信息 |
| `-v` | `--verbose` | 总是输出文件名头信息 |
| `-z` | `--zero-terminated` | 使用 NUL 字符作为行终止符，而不是换行符 |
| | `--help` | 显示帮助信息并退出 |
| | `--version` | 显示版本信息并退出 |

## 参数详细说明

### `-c, --bytes=[+]NUM`
显示每个文件的最后 NUM 个字节。NUM 可以是一个整数，也可以是以下单位之一：
- `b`：512 字节块
- `kB`：1000 字节
- `K`：1024 字节
- `MB`：1000000 字节
- `M`：1048576 字节
- `GB`：1000000000 字节
- `G`：1073741824 字节

如果 NUM 前有 `+`，则从第 NUM 个字节开始显示到文件末尾。

### `-n, --lines=[+]NUM`
显示每个文件的最后 NUM 行。默认值为 10。如果 NUM 前有 `+`，则从第 NUM 行开始显示到文件末尾。

### `-f, --follow[=how]`
跟踪模式，持续监控文件末尾的新增内容。how 参数可以是：
- `descriptor`：跟踪文件描述符（默认），即使文件被重命名或移动
- `name`：跟踪文件名，当文件被轮转或重新创建时自动切换

### `-F, --follow=name --retry`
等价于 `--follow=name --retry`，跟踪文件名并在文件不可用时持续重试。

### `--max-unchanged-stats=N`
使用 `--follow=name` 时，在文件大小连续 N 次检查不变后重新打开文件。默认值为 5。

### `--pid=PID`
与 `-f` 一起使用，当进程 PID 结束时自动终止 `tail`。

### `-q, --quiet, --silent`
安静模式，不输出文件名头信息。这是处理单个文件时的默认行为。

### `-v, --verbose`
详细模式，总是输出文件名头信息。这是处理多个文件时的默认行为。

### `-z, --zero-terminated`
使用 NUL 字符（\0）作为行终止符，而不是换行符（\n）。这对于处理包含换行符的文件名很有用。

## 使用示例

1. **显示文件的最后 10 行（默认）**
   ```bash
   tail file.txt
   ```

2. **显示文件的最后 20 行**
   ```bash
   tail -n 20 file.txt
   tail -20 file.txt
   ```

3. **显示文件的最后 100 个字节**
   ```bash
   tail -c 100 file.txt
   ```

4. **显示文件从第 15 行开始的内容**
   ```bash
   tail -n +15 file.txt
   ```

5. **跟踪文件的新增内容**
   ```bash
   tail -f /var/log/syslog
   ```

6. **跟踪文件名（处理日志轮转）**
   ```bash
   tail -F /var/log/syslog
   ```

7. **跟踪文件直到特定进程结束**
   ```bash
   tail -f --pid=1234 file.txt
   ```

8. **显示多个文件的最后 5 行，带文件名头**
   ```bash
   tail -n 5 -v file1.txt file2.txt
   ```

9. **使用 NUL 作为行终止符**
   ```bash
   tail -z -n 5 file.txt
   ```

10. **显示文件的最后 2 个块（每块 512 字节）**
    ```bash
    tail -c 2b file.txt
    ```
